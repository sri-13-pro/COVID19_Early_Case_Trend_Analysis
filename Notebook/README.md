# Notebook Documentation

## Overview

This directory contains the primary Jupyter Notebook used for the **COVID-19 Early Case Trend Analysis & Recovery Insights** project. The notebook presents the complete analytical workflow, from data loading and preprocessing to statistical analysis, machine learning, and business insights.

---

## Notebook

### `covid_case_analysis.ipynb`

The notebook includes the following sections:

1. Business Understanding
2. Library Imports
3. Data Loading
4. Data Understanding
5. Data Cleaning and Feature Engineering
6. Exploratory Data Analysis (EDA)
7. Statistical Analysis
8. Hypothesis Testing
9. Predictive Modeling

   * Linear Regression
   * Decision Tree Regressor
   * Random Forest Regressor
10. Model Comparison
11. Business Insights
12. Recommendations
13. Conclusion

---

## Requirements

Install the required Python packages before running the notebook:

```bash
pip install -r ../requirements.txt
```

---

## Execution Instructions

1. Open `covid_case_analysis.ipynb` using Jupyter Notebook or Visual Studio Code.
2. Ensure the dataset is available in the `../data/raw/` directory.
3. Run all notebook cells sequentially from top to bottom.
4. Processed datasets will be saved to `../data/processed/`.
5. Generated visualizations will be automatically stored in the corresponding folders under `../images/`.

---

## Outputs

The notebook generates:

* Cleaned dataset
* Exploratory Data Analysis visualizations
* Statistical analysis results
* Hypothesis testing outputs
* Machine learning model evaluations
* Model comparison metrics
* Business insights and recommendations
