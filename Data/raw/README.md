# Raw Data

## Overview

This directory contains the original dataset used for the **COVID-19 Early Case Trend Analysis & Recovery Insights** project.

The files in this folder represent the source data received before any preprocessing or transformation.

---

## Dataset

### `patient.csv`

The dataset contains patient-level COVID-19 records collected during the early stage of the outbreak.

The original data includes:

* Patient demographics
* Infection history
* Exposure details
* Timeline information
* Patient outcomes

---

## Purpose

The raw dataset serves as the primary source for all analyses performed in this project, including:

* Data Cleaning
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Machine Learning

---

## Important Notes

* Do **not** modify or overwrite the files in this directory.
* All preprocessing should be performed programmatically using the notebook or source code.
* This folder preserves the original dataset for reproducibility and auditing purposes.

---

## Data Source

COVID-19 Patient-Level Dataset provided as part of the HealthGuard Analytics Pvt. Ltd. case study.
