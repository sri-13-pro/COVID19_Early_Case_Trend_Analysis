# Processed Data

## Overview

This directory contains datasets generated after performing data cleaning, preprocessing, and feature engineering on the original COVID-19 patient records.

The processed datasets are used for analysis, visualization, statistical testing, and predictive modeling.

---

## Dataset

### `cleaned_patient_data.csv`

This dataset was generated after applying the complete preprocessing pipeline.

Processing steps include:

* Handling missing values
* Removing duplicate records
* Converting date columns
* Creating the **Age** feature
* Creating the **Age Group** feature
* Calculating **Recovery Duration**
* Standardizing categorical values
* Preparing the dataset for statistical analysis and machine learning

---

## Usage

The processed dataset is used throughout the project for:

* Exploratory Data Analysis (EDA)
* Descriptive Statistics
* Correlation Analysis
* Hypothesis Testing
* Linear Regression
* Decision Tree Regression
* Random Forest Regression
* Business Insights

---

## Data Integrity

This dataset is generated directly from the original data and should not be edited manually. Any updates or corrections should be made by rerunning the preprocessing pipeline.

---

## Output

The processed dataset provides a clean and analysis-ready version of the patient records, ensuring consistency and reproducibility across all stages of the project.
